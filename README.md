## Instalación de dependencias

Las dependencias de cada práctica se instalarán en un entorno virtual correspondiente dentro del directorio.

### Linux / Mac

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Windows

```
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
```

En el caso de que esté desactivada la ejecución de scripts utilizando windows powershell:

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
