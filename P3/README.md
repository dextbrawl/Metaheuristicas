# Consideraciones con la regresión simbólica

## 1 Número de puntos
Para el modelo A se han podido usar un número de puntos y una distancia Inter Par mayor puesto que la frontera que genera parece ser mayor. Sin embargo, con el modelo B debido a su mayor simpleza y menor tamaño de la frontera, ha debido usarse una menor distancia interpar, y menor número de puntos.


MODELO B

n_points = 15
th_interp_distance = 0.15


MODELO A

n_points = 35
th_interp_distance = 0.3

Generaciones = 1000
